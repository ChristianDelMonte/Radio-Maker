- OBJETIVOS e INTRODUCCIÓN:

El objetivo de esta mini-aplicación es la de proveer en tiempo real, sin cortes ni retrasos y de una manera sencilla e intuitiva; la hora y los datos climáticos de una zona especificada utilizando archivos de voz pregrabados por profesionales o personalizados por el usuario.

Para la reproducción horaria el sistema utiliza grabaciones previamente realizadas las cuales pueden mantenerse o bien, si el usuario lo desea, descartarse y utilizar grabaciones propias o realizadas por locutores autorizados. Mas adelante en esta guía le enseñaremos la manera de realizar esta tarea.

Para la reproducción de los datos climáticos el plug-in necesita conexión constante a la World Wide Web o a la Internet ya que extrae los datos como: temperatura y humedad valiéndose de sitios de reconocida trayectoria. RMVoice contiene una pequeña base de datos con lo necesario para extraer los datos climáticos de toda la Republica Argentina. Mas adelante en esta guía le enseñaremos como modificar y manipular esta base de datos para adaptarla a una zona especifica del planeta donde se encuentre utilizando el soft.

Este soft esta intentado para ser utilizado con ONLY Radiomaker solamente, por lo tanto no se recomienda el intento de uso con otras aplicaciones ajenas a la gama. El mismo viene por defecto configurado y adaptado para ser utilizado en dos idiomas: Español e Ingles; y puede ser adaptado a otros idiomas de una manera sencilla y practica. Mas adelante en esta guía le enseñaremos como personalizar el idioma para adaptarlo, también valido para cualquier aplicación y/o plug-in de la misma gama.

- REQUERIMIENTOS DE SISTEMA:

ONLY RMVoice requiere a ONLY Radiomaker versión 1.X instalado en su sistema para funcionar. No esta intentado para ser utilizado en otros sistemas o por otros medios.

  
     
  - INICIAR LA APLICACIÓN:  
     

1.1

  
  Una vez iniciado RadioMaker y cargado el plug-in como se muestra en la imagen 1.1. nos dirigiremos al menú "Plug-Ins" y luego a "RMPlug-Ins" para iniciar nuestro complemento "RMVoice". Si el mismo fue cargado correctamente al iniciar nuestra aplicación nos aparecerá enlistado como muestra la imagen 1.1 y podemos iniciarla simplemente haciendo clic en su nombre.  
     
  Una vez iniciada la aplicación nos aparecerá una ventana como la siguiente: 1.2  

1.2

  
  En la misma puede apreciarse la hora actual del sistema, la temperatura y la humedad del ambiente que en este caso aparece marcada como "N/A" debido a que el sistema no posee conexión a Internet o no pudo conectarse con el host proveedor de información climática. También podemos observar los datos de configuración básica del complemento que viene por defecto: los nombres de los archivos de audio para la reproducción horaria y climática en "Archivo de Hora","Archivo de minutos","Archivo de temperatura" y "Archivo de humedad"; el idioma actual del complemento: "Español"; la región para la extracción climática: "Salta, Argentina"; y el tipo de voz a utilizar: "Masculina" en este caso.

También encontramos varios botones que utilizaremos para probar que todo este bien configurado y funcione a la perfección:

El botón "reproducir hora" lo utilizaremos para reproducir la hora del momento y escuchar que se reproduzca correctamente una vez aplicados los cambios de configuración ya que de la misma manera en que se escucha una vez configurado se escuchara dentro del soft RadioMaker.

El botón "Temperatura" tiene el mismo propósito que el botón de reproducir hora con la diferencia de que se conectara a Internet, extraerá la información climática y nos la mostrara y luego la reproducirá como lo hizo con la hora anteriormente.

El botón de "Configuración" nos servirá para configurar los diferentes aspectos de nuestro complemento para que funcione correctamente conjuntamente a RadioMaker y de la manera que necesitamos.

  
     
  - CONFIGURACIÓN BÁSICA:  
     
  Ahora hacemos clic en el botón "Configuración" y nos aparecerá una ventana como la siguiente: 1.3  
1.3  
  RMVoice viene configurado por defecto para ser utilizado en dos idiomas: Español e Ingles. En selección de idioma puede seleccionar el que mas se adecue a sus necesidades. Mas adelante en las secciones "Archivos de Idioma" y "Personalización de Idiomas" le explicaremos como crear su propio archivo de idioma para adatarlo a sus necesidades.

En la sección de extracción de datos climáticos podrá seleccionar la provincia o estado en la que se encuentra dentro de la Republica Argentina, los mismos ya vienen cargados por defecto en el sistema, pero podrá personalizar una zona especifica si lo desea mas adelante haciendo clic en "Nuevo". En la sección "Configuración de URL climática" le explicaremos como crear su propia zona de extracción climática personalizada.

Seleccione el modo de visualización de temperatura que desea entre "Grados Fahrenheit" o "Grados Centígrados" según se adecue a sus necesidades.

En tiempo de actualización on-line deberá setear el tiempo de espera (en segundos) que el sistema realizara antes de intentar una conexión a Internet para extraer los datos climáticos. Recuerde que debe poseer una conexión a Internet constante o de banda ancha constante para poder utilizar esta opción.

A continuación podrá seleccionar entre tres tipos de voces para la reproducción de la hora o el clima. Por defecto vienen dos configurados ya previamente en el sistema: "voz masculina" o "voz femenina", pero puede utilizar un diferente tipo de voz diferente simplemente seleccionando "personalizada" y buscando los archivos de audio correspondientes haciendo clic en los botones "/..." que se encuentran mas abajo.

Para finalizar haga clic en "Aplicar" para aplicar los cambios realizados y luego haga clic en "Cerrar".

  - ARCHIVOS DE IDIOMA:  
     
  Dentro de la carpeta "Plugins", a su vez dentro de la carpeta de instalación de la aplicación encontraremos una serie de archivos necesarios para la personalización del idioma y otros como muestra la figura 1.5. Le recomendamos hacer una copia de seguridad de estos archivos antes de realizar cualquier modificación en los mismos. Para ello cree una carpeta llamada "Backup" y copie todos los archivos dentro de esa carpeta para luego poder restaurarlos a su ubicación original en caso de ocurrir algún inconveniente.  
1.5  
  El archivo llamado "RMVoice.Localize.cfg" contiene la información sobre los idiomas y los archivos de idioma a utilizar, este es el archivo que deberá modificar para agregar un nuevo archivo de idioma a la aplicación.

Los archivos "RMVoice.ES.lng" y "RMVoice.EN.lng" son los archivos de idioma de la aplicación en "Español" e "Ingles" respectivamente, estos archivos podremos usarlos como guía para otros idiomas nuevos.
